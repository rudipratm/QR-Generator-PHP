Aplikasi QR Generator v.1.0
Oleh Rudi Pratama

Aplikasi ini dibangun menggunakan bahasa PHP dan library phpqrcore. Aplikasi ini berfungsi untuk meng-generate sebuah gambar QR Code dari teks yang dimasukkan.

Pengaturan:
1. Ukuran gambar ada pada index.php bagian QRcode::png($text, false, QR_ECLEVEL_H, 10, 3);
2. Jika kurang paham dengan php, sebaiknya jangan mengubah apapun didalam folder phpqrcode.
